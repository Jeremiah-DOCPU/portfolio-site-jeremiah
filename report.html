Portfolio Development Report

Introduction

In this project, I developed a personal portfolio website to showcase my work, skills and progress as a Computer Science student. The purpose of this project was to take my existing knowledge of HTML, CSS and JavaScript and put it to use. I wanted to make something that reflected my personal growth.

Planning

Before starting the development, I planned the structure of my website by deciding what pages I want and would be required. I decided to add a home page, a projects page, a contact page, a video demo page and a report page.This allowed me to clearly organise my content and ensure that users could easily navigate through the website.

I also thought about how users would interact with the site. To ensure that users do not get lost when transitioning between various sections, I ensured the navigation bar of all pages was consistent.

Design

For the design, I aimed to create a modern and clean layout. I used a consistent colour scheme, spacing and typography across the site to improve the overall user experience.

I employed CSS Flexbox and Grid to arrange the layout of various sections like the Navigation Bar, Skills section, and Projects page.

To improve usability, I used clear headings and easy-to-read copy so that the hierarchy was understandable.

Development

I developed the website using HTML for structure, CSS for styling and JavaScript for interactivity.

One of the main features I implemented was the contact form. Instead of using a server-side solution, I used JavaScript to create a mailto link that opens the user’s email client with a pre-filled message.

I also created a projects page using a grid layout and cards, which made the content more organised and visually appealing.

Testing

During development, I tested the website regularly in the browser to ensure all links worked correctly and the layout appeared as expected.

I also checked that the form inputs worked properly and that the email client opened correctly when submitting the contact form.

Challenges

One of the main challenges I faced was making sure all parts of the website were connected properly, especially linking pages and ensuring the navigation worked on every page.

Another challenge was structuring the CSS so that it remained organised while the website became more complex.

Reflection

Overall, this project helped me improve my understanding of web development. I gained more confidence in writing HTML, styling with CSS and adding basic interactivity using JavaScript.

If I had more time, I would improve the website further by adding more advanced JavaScript features and improving the responsiveness for mobile devices.

This project has been valuable in helping me understand how a complete website is planned, built and refined over time.
